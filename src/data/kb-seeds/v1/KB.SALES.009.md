# Maklergespräch Script

## Ziel des Gesprächs
- Vollständige Objektinformationen beschaffen
- Verkäufermotivation verstehen
- Verhandlungsspielraum ausloten
- Professionellen Eindruck hinterlassen

---

## Phase 1: Eröffnung (2 Minuten)

### Begrüßung
```
"Guten Tag, [Name]. Vielen Dank, dass Sie sich Zeit nehmen.
Ich bin [Name], Kapitalanleger mit Fokus auf [Objekttyp/Region].
Das Objekt in [Adresse] interessiert mich sehr."
```

### Qualifizierung (kurz)
```
"Ich kaufe regelmäßig Kapitalanlageimmobilien und suche
aktuell [Kriterien: z.B. vermietete Wohnungen, 80-150k].
Können Sie mir mehr zum Objekt erzählen?"
```

---

## Phase 2: Objektinformationen (10 Minuten)

### Grunddaten
```
"Lassen Sie uns mit den Basics starten:"
- Wie groß ist die Wohnfläche genau?
- Welches Baujahr hat das Gebäude?
- Wie viele Einheiten hat das Haus insgesamt?
- Wie hoch ist der Miteigentumsanteil?
```

### Zustand & Ausstattung
```
"Zum Zustand:"
- Wann wurde zuletzt renoviert?
- Wie alt sind Heizung, Fenster, Dach?
- Gibt es bekannte Mängel oder Sanierungsstau?
- Welche Ausstattung ist vorhanden (Einbauküche, Balkon)?
```

### Mietverhältnis (bei vermietet)
```
"Zur aktuellen Vermietung:"
- Wie hoch ist die Kaltmiete?
- Seit wann wohnt der Mieter?
- Wie ist das Zahlungsverhalten?
- Ist die Miete unter Marktniveau?
```

---

## Phase 3: WEG-Informationen (5 Minuten)

```
"Zur Eigentümergemeinschaft:"
- Wie hoch ist das monatliche Hausgeld?
- Wie hoch ist die Instandhaltungsrücklage?
- Sind Sonderumlagen geplant oder beschlossen?
- Gibt es laufende Rechtsstreitigkeiten?
- Wer verwaltet die WEG?
```

---

## Phase 4: Verkäufermotivation (3 Minuten)

### Weiche Fragen
```
"Darf ich fragen:"
- Warum wird verkauft?
- Wie lange ist das Objekt schon im Besitz?
- Gibt es einen Zeitdruck beim Verkauf?
```

### Interpretation

| Antwort | Bedeutung |
|---------|-----------|
| "Erbschaft" | Oft verhandlungsbereit |
| "Portfolio bereinigen" | Schneller Abschluss wichtig |
| "Altersvorsorge" | Emotionale Bindung möglich |
| "Umzug" | Zeitdruck vorhanden |

---

## Phase 5: Verhandlungsspielraum (2 Minuten)

```
"Zum Kaufpreis:"
- Ist der Preis verhandelbar?
- Wie lange ist das Objekt schon inseriert?
- Gab es schon andere Interessenten/Angebote?
- Gibt es Präferenzen (schnelle Abwicklung vs. Preis)?
```

---

## Phase 6: Unterlagen & Next Steps (2 Minuten)

### Unterlagen anfordern
```
"Können Sie mir bitte folgende Unterlagen zusenden:"
- Grundbuchauszug
- Teilungserklärung
- Hausgeldabrechnungen (3 Jahre)
- Protokolle der letzten 3 EV
- Mietvertrag
- Energieausweis
- Grundriss
```

### Nächster Schritt
```
"Vielen Dank für die Informationen.
Sobald ich die Unterlagen geprüft habe, melde ich mich
für einen Besichtigungstermin. Wann passt es Ihnen am besten?"
```

---

## Nachbereitung

### Sofort dokumentieren
- [ ] Alle genannten Zahlen notieren
- [ ] Red Flags festhalten
- [ ] Eindruck vom Makler
- [ ] Nächste Schritte terminieren

### Follow-up
- [ ] Fehlende Unterlagen nachfordern
- [ ] Besichtigungstermin bestätigen
- [ ] Eigene Recherche zur Lage

---

## Red Flags im Gespräch

| Signal | Mögliche Bedeutung |
|--------|-------------------|
| Ausweichende Antworten | Probleme vorhanden |
| Druck zum schnellen Abschluss | Versteckte Mängel? |
| Keine Unterlagen verfügbar | Unprofessionell |
| Hohe Preisvorstellung ohne Begründung | Überteuert |
| WEG-Infos unbekannt | Schlecht vorbereitet |
