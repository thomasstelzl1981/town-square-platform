# Due Diligence Workflow

## Überblick

Die Due Diligence ist die systematische Prüfung einer Immobilie vor dem Kauf. Sie minimiert Risiken und schafft Klarheit über den wahren Zustand und Wert.

---

## Phase 1: Dokumentenprüfung (Desk Review)

### Rechtliche Dokumente
- [ ] Grundbuchauszug (nicht älter als 3 Monate)
- [ ] Teilungserklärung + Gemeinschaftsordnung (bei WEG)
- [ ] Baulastenverzeichnis
- [ ] Altlastenkataster

### Wirtschaftliche Dokumente
- [ ] Mietverträge (alle Einheiten)
- [ ] Betriebskostenabrechnungen (3 Jahre)
- [ ] Hausgeldabrechnungen (3 Jahre, bei WEG)
- [ ] Wirtschaftsplan aktuell

### Technische Dokumente
- [ ] Energieausweis
- [ ] Grundrisse + Flächenberechnung
- [ ] Baugenehmigung
- [ ] Modernisierungsnachweise

---

## Phase 2: WEG-Prüfung (nur Eigentumswohnung)

- [ ] Protokolle Eigentümerversammlungen (3 Jahre)
- [ ] Instandhaltungsrücklage (Höhe, Entwicklung)
- [ ] Beschlossene Sonderumlagen
- [ ] Offene Rechtsstreitigkeiten
- [ ] Verwaltungsvertrag

**Red Flags:**
- Rücklage < 20 €/m² Gesamtfläche
- Häufige Sonderumlagen
- Streitigkeiten mit hohem Streitwert

---

## Phase 3: Besichtigung (On-Site)

### Gebäude außen
- [ ] Dach (Zustand, Alter)
- [ ] Fassade (Risse, Feuchtigkeit)
- [ ] Fenster (Alter, Dichtigkeit)
- [ ] Keller (Feuchtigkeit, Geruch)
- [ ] Heizungsanlage (Typ, Alter, Wartung)

### Wohnung/Einheit
- [ ] Böden, Wände, Decken
- [ ] Sanitär (Zustand, Leitungen)
- [ ] Elektrik (FI-Schalter, Steckdosen)
- [ ] Fenster, Türen
- [ ] Balkon/Terrasse

### Umfeld
- [ ] Parkplätze, Zufahrt
- [ ] Nachbarschaft (Lärm, Zustand)
- [ ] Infrastruktur (ÖPNV, Einkaufen)

---

## Phase 4: Mieterprüfung (vermietete Objekte)

- [ ] Mietverträge analysieren (Laufzeit, Kündigungsfristen)
- [ ] Zahlungshistorie (Rückstände?)
- [ ] Mieterhöhungspotenzial
- [ ] Mieterstruktur (Alter, Bonität)

---

## Phase 5: Bewertung & Entscheidung

| Prüfpunkt | Status | Notiz |
|-----------|--------|-------|
| Rechtlich sauber | ✓/✗ | |
| Wirtschaftlich plausibel | ✓/✗ | |
| Technisch akzeptabel | ✓/✗ | |
| Mieter stabil | ✓/✗ | |
| Preis marktgerecht | ✓/✗ | |

**Entscheidung:** Kaufen / Nachverhandeln / Ablehnen

---

## Nächste Schritte

Bei positivem Ergebnis:
1. Kaufpreisverhandlung
2. Finanzierungszusage einholen
3. Notartermin vereinbaren
