# Besichtigungs-Checkliste

## Fotoliste (systematisch)

### Außenbereich
- [ ] Gesamtansicht Gebäude (frontal)
- [ ] Fassade (alle Seiten)
- [ ] Dach (soweit sichtbar)
- [ ] Eingangsbereich/Hausflur
- [ ] Klingelanlage/Briefkästen
- [ ] Mülltonnenplatz
- [ ] Parkplätze/Garage
- [ ] Garten/Grünflächen

### Treppenhaus/Gemeinschaftsflächen
- [ ] Treppenhaus (mehrere Etagen)
- [ ] Aufzug (falls vorhanden)
- [ ] Kellerflur
- [ ] Waschküche
- [ ] Fahrradkeller
- [ ] Dachboden (falls zugänglich)

### Wohnung/Einheit
- [ ] Eingangsbereich/Flur
- [ ] Wohnzimmer (Übersicht + Details)
- [ ] Küche (inkl. Geräte, Anschlüsse)
- [ ] Badezimmer (Armaturen, Fliesen)
- [ ] Schlafzimmer (alle)
- [ ] Balkon/Terrasse
- [ ] Fenster (Innenansicht)
- [ ] Heizkörper/Thermostate
- [ ] Sicherungskasten
- [ ] Keller (wenn zugehörig)

### Detailfotos (Mängel)
- [ ] Risse in Wänden/Decken
- [ ] Feuchtigkeitsflecken
- [ ] Schimmel
- [ ] Beschädigte Böden
- [ ] Defekte Fenster/Türen
- [ ] Veraltete Elektrik

## Fragenkatalog

### An Makler/Verkäufer

**Zum Gebäude:**
1. Baujahr? Letzte Sanierungen?
2. Wann wurde das Dach zuletzt erneuert?
3. Alter und Zustand der Heizung?
4. Gab es Wasserschäden?
5. Sind Sanierungen geplant?

**Zur Wohnung:**
1. Wann wurde zuletzt renoviert?
2. Gibt es bekannte Mängel?
3. Ist die Wohnung unterkellert?
4. Wie ist die Ausrichtung (Himmelsrichtung)?
5. Wie hoch sind die Nebenkosten?

**Zum Mieter (falls vermietet):**
1. Wie lange wohnt der Mieter schon?
2. Pünktliche Mietzahlung?
3. Wie ist das Verhältnis?
4. Sind Mieterhöhungen möglich?

**Zur WEG:**
1. Wie viele Eigentümer gibt es?
2. Wie hoch ist die Rücklage?
3. Sind Sonderumlagen geplant?
4. Gibt es Streitigkeiten?

### Beobachten (nicht fragen)

- Zustand des Treppenhauses
- Nachbarschaft (Sauberkeit, Lärm)
- Parksituation
- Gerüche
- Lichteinfall
