# Haushaltsrechnung verstehen

> **Hinweis:** Diese Information dient nur zur allgemeinen Orientierung. Jede Bank rechnet etwas anders.

---

## Was prüft die Bank?

Die Haushaltsrechnung zeigt, ob Sie sich die Kreditrate leisten können.

```
Einnahmen - Ausgaben - neue Rate ≥ Puffer
```

---

## Einnahmen (was die Bank anerkennt)

| Position | Anrechnung |
|----------|------------|
| Netto-Gehalt | 100% |
| 13. Gehalt/Bonus | 50-80% |
| Mieteinnahmen (bestehend) | 70-80% |
| Mieteinnahmen (neu) | 50-70% |
| Kindergeld | 100% |
| Unterhalt (erhalten) | 100% |
| Nebenjob | 0-50% |

**Vorsichtig:** Banken rechnen konservativ!

---

## Ausgaben (was die Bank ansetzt)

| Position | Typischer Wert |
|----------|----------------|
| Lebenshaltung (1 Person) | 750-900 € |
| Lebenshaltung (Familie 4P) | 1.200-1.500 € |
| Pro Kind zusätzlich | 200-300 € |
| PKW | 250-350 € |
| Bestehende Kredite | Tatsächliche Rate |
| Unterhalt (zahlen) | Tatsächlicher Betrag |
| Warmmiete (falls nicht Eigentum) | Tatsächliche Miete |

---

## Rechenbeispiel

### Familie, 2 Kinder, Bestandsimmobilie

**Einnahmen:**
| Position | Betrag |
|----------|--------|
| Netto-Gehalt | 4.500 € |
| Kindergeld | 500 € |
| **Summe** | **5.000 €** |

**Ausgaben (Bank):**
| Position | Betrag |
|----------|--------|
| Lebenshaltung Familie | 1.400 € |
| PKW | 300 € |
| Bestehendes Darlehen | 500 € |
| **Summe** | **2.200 €** |

**Verfügbar:**
```
5.000 € - 2.200 € = 2.800 € für neue Rate
```

**Neue Finanzierung:**
- Maximal tragbare Rate: ca. 2.500 € (Puffer!)
- Entspricht bei 4% Zins + 2% Tilgung: ca. 500.000 € Darlehen

---

## Kapitaldienstfähigkeit

Die Bank prüft auch mit Stresszinsen:
- Was passiert bei 6% statt 4%?
- Bleibt die Rate tragbar?

---

## Tipps

1. **Aufräumen:** Kleine Kredite ablösen
2. **Dispokredite:** Zurückführen
3. **Nebeneinkünfte:** Dokumentieren
4. **Eigenkapital:** Nachweisbar halten
