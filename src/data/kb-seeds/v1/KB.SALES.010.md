# Lead-Qualifizierung Checkliste

## Übersicht

Ein qualifizierter Lead ist ein potenzieller Käufer mit hoher Abschlusswahrscheinlichkeit. Diese Checkliste hilft bei der systematischen Bewertung.

---

## Scoring-Modell (BANT+)

### B — Budget (0–25 Punkte)

| Kriterium | Punkte |
|-----------|--------|
| Eigenkapital vorhanden (> 20% Ziel-KP) | +10 |
| Finanzierungszusage liegt vor | +10 |
| Budget passt zu Objektangebot | +5 |
| Budget unklar/nicht genannt | 0 |

### A — Authority (0–20 Punkte)

| Kriterium | Punkte |
|-----------|--------|
| Alleiniger Entscheider | +15 |
| Partner eingebunden, beide überzeugt | +10 |
| Partner noch nicht involviert | +5 |
| Entscheidung durch Dritte (Eltern, Berater) | 0 |

### N — Need (0–25 Punkte)

| Kriterium | Punkte |
|-----------|--------|
| Klares Anlageziel definiert (Rendite, Alter) | +10 |
| Spezifische Suchkriterien | +10 |
| Dringlichkeit vorhanden (Zeithorizont < 6 Monate) | +5 |
| "Schaue mich nur um" | 0 |

### T — Timeline (0–20 Punkte)

| Kriterium | Punkte |
|-----------|--------|
| Kaufbereit innerhalb 1–3 Monate | +20 |
| Kaufbereit innerhalb 3–6 Monate | +15 |
| Kaufbereit innerhalb 6–12 Monate | +10 |
| Kein konkreter Zeitrahmen | +5 |

### + — Engagement (0–10 Punkte)

| Kriterium | Punkte |
|-----------|--------|
| Antwortet schnell (< 24h) | +3 |
| Hat bereits Objekte besichtigt | +3 |
| Stellt detaillierte Fragen | +2 |
| Hat Unterlagen vorbereitet | +2 |

## Gesamtbewertung

| Score | Kategorie | Priorität | Aktion |
|-------|-----------|-----------|--------|
| 80–100 | A-Lead (Hot) | Höchste | Sofort Termin |
| 60–79 | B-Lead (Warm) | Hoch | Innerhalb 48h |
| 40–59 | C-Lead (Lukewarm) | Mittel | Nurturing |
| < 40 | D-Lead (Cold) | Niedrig | Automatisierung |

## Qualifizierungsfragen

### Budget ermitteln
```
"Haben Sie sich schon mit der Finanzierung beschäftigt?"
"Welches Kaufpreissegment schwebt Ihnen vor?"
"Haben Sie bereits Eigenkapital reserviert?"
```

### Entscheidungsfähigkeit prüfen
```
"Treffen Sie die Kaufentscheidung alleine?"
"Ist Ihr Partner/Ihre Partnerin auch interessiert?"
"Gibt es weitere Personen, die involviert sein sollten?"
```

### Bedarf verstehen
```
"Was ist Ihr Hauptziel bei der Kapitalanlage?"
"Welche Rendite erwarten Sie?"
"Welche Objekte haben Sie bisher angeschaut?"
```

### Zeitrahmen klären
```
"Bis wann möchten Sie idealerweise gekauft haben?"
"Gibt es einen konkreten Anlass für die Suche?"
"Was müsste passieren, damit Sie sich entscheiden?"
```

---

## Disqualifizierungs-Kriterien

### Sofortige Disqualifikation
- [ ] Kein Budget / unrealistische Erwartungen
- [ ] Keine Entscheidungsfähigkeit (z.B. "muss erst fragen")
- [ ] Kein echtes Interesse (nur Informationssammlung)
- [ ] Außerhalb des Zielgebiets

### Orange Flags (beobachten)
- [ ] Sehr lange Timeline (> 12 Monate)
- [ ] Unklare Finanzierung
- [ ] Widersprüchliche Aussagen
- [ ] Schlechte Erreichbarkeit

## Lead-Status im CRM

| Status | Definition | Nächste Aktion |
|--------|------------|----------------|
| Neu | Gerade eingegangen | Qualifizieren (< 24h) |
| Qualifiziert | BANT geprüft | Termin vereinbaren |
| In Bearbeitung | Im Verkaufsprozess | Follow-up |
| Verhandlung | Konkretes Objekt | Abschluss vorbereiten |
| Gewonnen | Kaufvertrag | Übergabe |
| Verloren | Kein Abschluss | Grund dokumentieren |
| Nurturing | Noch nicht bereit | Automatisierung |
