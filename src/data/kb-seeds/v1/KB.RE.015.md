# Exposé-Analyse: Playbook

## Schritt 1: Erste Einordnung (2 Minuten)

### Schnell-Check
- [ ] Preis und Lage plausibel?
- [ ] Rendite realistisch (> 8% = kritisch prüfen)?
- [ ] Baujahr und Zustand passen zusammen?
- [ ] Fotos wirken aktuell?

### Erste Red Flags
| Signal | Bedeutung |
|--------|-----------|
| "Renovierungsbedarf" | Erhebliche Kosten |
| "Ideal für Handwerker" | Schlechter Zustand |
| "Kapitalanleger" | Mieter eventuell problematisch |
| Keine Innenfotos | Etwas wird versteckt |
| "Verkauf im Bieterverfahren" | Preis wird steigen |

---

## Schritt 2: Zahlen prüfen (5 Minuten)

### Flächenangaben
```
Wohnfläche ≠ Nutzfläche ≠ Grundfläche
```

- [ ] Wohnfläche nach WoFlV?
- [ ] Balkone/Terrassen korrekt berechnet (50%)?
- [ ] Keller/Dachboden separat ausgewiesen?

### Mietangaben
- [ ] Kaltmiete oder Warmmiete?
- [ ] Ist-Miete vs. Soll-Miete?
- [ ] Mieterhöhungspotenzial?
- [ ] Leerstand erwähnt?

### Rendite-Check
```
Bruttomietrendite = (Jahreskaltmiete / Kaufpreis) × 100
```

| Angegeben | Nachrechnen! |
|-----------|--------------|
| 5% Rendite | Nebenkosten fehlen oft |
| "X-fache Miete" | Faktor 20 = 5% Brutto |

---

## Schritt 3: Unterlagen-Check

### Was MUSS angefordert werden?
- [ ] Grundbuchauszug (aktuell)
- [ ] Energieausweis
- [ ] Grundriss mit Maßen
- [ ] Mietverträge (bei vermietet)
- [ ] Nebenkostenabrechnungen (3 Jahre)

### Bei WEG zusätzlich:
- [ ] Teilungserklärung
- [ ] Protokolle (3 Jahre)
- [ ] Wirtschaftsplan
- [ ] Hausgeldabrechnung
- [ ] Rücklagenstand

---

## Schritt 4: Nachfragen formulieren

### Standard-Fragenkatalog

**Zum Objekt:**
1. Warum wird verkauft?
2. Wie lange im Besitz des Verkäufers?
3. Welche Renovierungen wurden durchgeführt (wann)?
4. Gibt es bekannte Mängel?
5. Wie alt sind Heizung, Dach, Fenster?

**Zum Mieter (wenn vermietet):**
1. Wie lange wohnt der Mieter schon dort?
2. Gibt es Mietrückstände?
3. Wurde die Miete schon angepasst?
4. Wie ist das Verhältnis zum Mieter?

**Zur WEG:**
1. Sind Sonderumlagen geplant?
2. Gibt es Rechtsstreitigkeiten?
3. Wie ist die Eigentümerstruktur (Selbstnutzer/Vermieter)?
4. Wer ist die Hausverwaltung?

| Kriterium | Bewertung (1-5) | Gewichtung |
|-----------|-----------------|------------|
| Lage | | 25% |

| Rendite | | 20% |

**Score < 3:** Ablehnen
**Score 3-4:** Besichtigen
**Score > 4:** Priorisieren

## Häufige Exposé-Tricks

| Trick | Realität |
|-------|----------|
| "Bezugsfrei" bei vermietetem Objekt | Kündigung geplant/gehofft |
| Nur Außenfotos | Innen problematisch |
| "Renditeobjekt" | Schlechter Mieter möglich |
| Sehr viele Fotos von Details | Gesamtzustand mäßig |
| "Provisionsfrei" | Im Kaufpreis enthalten |
