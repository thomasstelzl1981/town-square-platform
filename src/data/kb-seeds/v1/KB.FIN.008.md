# Refinanzierung Playbook

> **Hinweis:** Diese Information dient nur zur allgemeinen Orientierung. Für verbindliche Auskünfte wenden Sie sich bitte an einen Finanzierungsberater.

---

## Wann Refinanzierung prüfen?

### Trigger-Events
- [ ] Zinsbindung läuft aus (in 6–24 Monaten)
- [ ] 10 Jahre seit Darlehensbeginn (Sonderkündigungsrecht)
- [ ] Zinsniveau deutlich gesunken
- [ ] Wertsteigerung der Immobilie (besserer LTV)
- [ ] Verbesserte Bonität

---

## Schritt 1: Ist-Situation analysieren

### Aktuelle Konditionen erfassen

| Parameter | Aktuell |
|-----------|---------|
| Restschuld | € |
| Aktueller Zinssatz | % |
| Restlaufzeit Zinsbindung | Monate |
| Monatliche Rate | € |
| Sondertilgungsoption | Ja/Nein |

---

## Schritt 2: Kosten einer Ablösung kalkulieren

### Vor Ende der Zinsbindung

```
Vorfälligkeitsentschädigung (VFE)
= Entgangener Zinsertrag der Bank
→ Kann mehrere Tausend Euro betragen!
```

**Berechnung anfordern:** Bank muss VFE auf Anfrage berechnen.

### Nach 10 Jahren (§ 489 BGB)

```
Keine VFE!
Nur 6 Monate Kündigungsfrist
```

---

## Schritt 3: Break-Even berechnen

### Formel (vereinfacht)

```
Break-Even (Monate) = Gesamtkosten / Monatliche Ersparnis
```

**Beispiel:**

| Position | Wert |
|----------|------|
| VFE | 5.000 € |
| Notarkosten (falls nötig) | 500 € |
| Bearbeitungsgebühren | 200 € |
| **Gesamtkosten** | **5.700 €** |
| Alte Rate | 1.200 € |
| Neue Rate | 1.050 € |
| **Ersparnis/Monat** | **150 €** |
| **Break-Even** | **38 Monate** |

**Entscheidung:**
- Verbleibende Laufzeit > Break-Even → Refinanzierung sinnvoll
- Verbleibende Laufzeit < Break-Even → Nicht sinnvoll

---

## Schritt 5: Entscheidung

### Pro Refinanzierung
✅ Break-Even < 50% der Restlaufzeit
✅ Signifikante Zinsersparnis (> 0,5%)
✅ Bessere Konditionen (Sondertilgung, Flexibilität)
✅ Kein VFE (nach 10 Jahren)

### Contra Refinanzierung
❌ Hohe VFE
❌ Geringe Restschuld
❌ Kurze Restlaufzeit
❌ Aufwand übersteigt Nutzen

---

## Checkliste Ablöseprozess

- [ ] VFE-Berechnung anfordern
- [ ] Neue Finanzierung zusagen lassen
- [ ] Kündigungsfrist beachten (meist 3–6 Monate)
- [ ] Grundschuldabtretung prüfen (spart Notar)
- [ ] Zeitpunkt koordinieren (nahtloser Übergang)

---

## Grundschuldabtretung

```
Statt neuer Grundschuld:
Abtretung der bestehenden Grundschuld an neue Bank
→ Spart Notar- und Grundbuchkosten (ca. 0,5–1% der Grundschuld)
```
