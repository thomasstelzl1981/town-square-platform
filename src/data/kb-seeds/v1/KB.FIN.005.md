# Finanzierungs-Dokumentpaket

> Vollständige Checkliste für die Kreditanfrage bei der Bank.

## Persönliche Unterlagen

### Identifikation
- [ ] Personalausweis/Reisepass (Kopie)

### Einkommensnachweise (Angestellt)
- [ ] Letzte 3 Gehaltsabrechnungen
- [ ] Arbeitsvertrag

### Einkommensnachweise (Selbstständig)
- [ ] Letzte 3 Einkommensteuerbescheide
- [ ] Letzte 3 Jahresabschlüsse/BWA
- [ ] Einnahmen-Überschuss-Rechnung

- [ ] Bausparverträge
