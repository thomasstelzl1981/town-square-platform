# E-Mail-Vorlage: Bank-Unterlagenliste

## Verwendungszweck
Erste Anfrage bei der Bank oder Antwort auf Unterlagen-Anforderung.

---

## Vorlage

**Betreff:** Finanzierungsanfrage - [Vorname Nachname] - [Objekt Adresse]

---

Sehr geehrte Damen und Herren, / Sehr geehrte/r [Anrede + Name],

ich interessiere mich für ein Immobilieninvestment und möchte Sie um ein Finanzierungsangebot bitten.

**Zum Objekt:**
- Adresse: [Straße, PLZ Ort]
- Kaufpreis: [X] €
- Art: [ETW / MFH / EFH]
- Nutzung: Vermietung als Kapitalanlage

**Zur Finanzierung:**
- Gewünschter Darlehensbetrag: [X] €
- Eigenkapitaleinsatz: [X] €
- Gewünschte Zinsbindung: [X] Jahre
- Anfängliche Tilgung: [X] %
- Sondertilgungsoption erwünscht: Ja / Nein

**Zu meiner Person:**
In der Anlage finden Sie:
- [ ] Selbstauskunft (ausgefüllt)
- [ ] Letzte 3 Gehaltsabrechnungen
- [ ] Letzter Einkommensteuerbescheid
- [ ] Kontoauszüge (EK-Nachweis)
- [ ] Objektunterlagen (Exposé, Grundriss, Energieausweis)

Für Rückfragen stehe ich gerne zur Verfügung. Über ein zeitnahes Angebot würde ich mich freuen.

Mit freundlichen Grüßen

[Vorname Nachname]
[Telefon]
[E-Mail]

**Anlagen:**
- Selbstauskunft_[Name].pdf
- Gehaltsabrechnungen.pdf
- Steuerbescheid_[Jahr].pdf
- EK_Nachweis.pdf
- Objektunterlagen.pdf

---

## Tipps

1. **Vollständigkeit:** Je vollständiger, desto schneller die Bearbeitung
2. **PDF:** Alle Dokumente als PDF, keine Fotos
3. **Benennung:** Klare Dateinamen verwenden
4. **Telefon:** Erreichbarkeit sicherstellen
