# Objektprüfung vor Kauf: Unterlagenliste

## Grundlegende Unterlagen

### Kaufvertragsentwurf
- [ ] Vollständiger Entwurf vom Notar
- [ ] Übergabetermin definiert
- [ ] Zahlungsmodalitäten klar

### Grundbuchauszug
- [ ] Aktuell (nicht älter als 3 Monate)
- [ ] Eigentümerverhältnisse
- [ ] Abteilung II: Lasten (Wegerechte, Wohnrechte)
- [ ] Abteilung III: Grundschulden

### Flurkarte/Lageplan
- [ ] Grundstücksgrenzen
- [ ] Zufahrt/Wegerecht

---

## Gebäude & Technik

### Baupläne
- [ ] Grundrisse aller Geschosse
- [ ] Schnitte
- [ ] Ansichten

### Baugenehmigung
- [ ] Vorhanden und vollständig
- [ ] Nutzung entspricht Genehmigung

### Energieausweis
- [ ] Gültiger Ausweis (10 Jahre)
- [ ] Typ: Bedarfs- oder Verbrauchsausweis
- [ ] Energieeffizienzklasse

### Wartungsnachweise
- [ ] Heizung (letzte Wartung)
- [ ] Aufzug (falls vorhanden)
- [ ] Feuerlöscher/Brandschutz

---

## WEG (bei Eigentumswohnung)

### Teilungserklärung
- [ ] Vollständig mit Gemeinschaftsordnung
- [ ] Sondernutzungsrechte
- [ ] Stimmrechtsverteilung

### Protokolle Eigentümerversammlung
- [ ] Letzte 3 Jahre
- [ ] Beschlüsse über Sanierungen
- [ ] Sonderumlagen

### Wirtschaftsplan & Hausgeldabrechnung
- [ ] Aktueller Wirtschaftsplan
- [ ] Letzte 3 Abrechnungen
- [ ] Höhe der Instandhaltungsrücklage

---

## Vermietung

### Mietverträge
- [ ] Alle aktuellen Verträge
- [ ] Mieterhöhungshistorie
- [ ] Staffel-/Indexklauseln

### Mieterakte
- [ ] Zahlungshistorie
- [ ] Kaution (Höhe, Anlage)
- [ ] Kommunikation

### Nebenkostenabrechnungen
- [ ] Letzte 3 Jahre
- [ ] Nachzahlungen/Guthaben

---

## Finanzen

### Kaufpreiszusammensetzung
- [ ] Aufschlüsselung (Grund/Gebäude/Inventar)
- [ ] Relevant für AfA

### Mietaufstellung
- [ ] IST-Mieten (Kalt + Nebenkosten)
- [ ] Mietpotenzial bei Neuvermietung
