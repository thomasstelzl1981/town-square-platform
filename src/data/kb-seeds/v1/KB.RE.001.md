# Kapitalanlage Immobilie: Grundbegriffe

## Rendite-Begriffe

### Bruttomietrendite

```
Bruttomietrendite = (Jahreskaltmiete / Kaufpreis) × 100
```

Beispiel: 12.000 € Jahresmiete / 200.000 € Kaufpreis = **6,0%**

**Wichtig:** Enthält keine Nebenkosten oder Aufwendungen.

---

### Nettomietrendite

```
Nettomietrendite = ((Jahreskaltmiete - Bewirtschaftungskosten) / Kaufpreis) × 100
```

Bewirtschaftungskosten: Verwaltung, Instandhaltungsrücklage, nicht umlegbare NK

**Realistischer** als Bruttorendite, typisch 1-2% niedriger.

---

### Eigenkapitalrendite

```
EK-Rendite = (Jahresüberschuss nach Zins / Eigenkapital) × 100
```

Berücksichtigt den Hebel durch Fremdfinanzierung.

---

## Cashflow-Begriffe

### Cashflow vor Steuern

```
Kaltmiete
- Hausgeld (nicht umlegbar)
- Zins + Tilgung
- Sonstige Kosten
= Cashflow vor Steuern
```

### Cashflow nach Steuern

Berücksichtigt zusätzlich:
- AfA (positiv)
- Werbungskosten (positiv)
- Zu versteuernde Mieteinnahmen (negativ)

---

## Weitere wichtige Begriffe

| Begriff | Bedeutung |
|---------|-----------|
| **Kaufnebenkosten** | Notar, Grundbuch, GrESt, Makler |
| **Beleihungsauslauf (LTV)** | Kredit / Kaufpreis inkl. NK |
| **Mietmultiplikator** | Kaufpreis / Jahresmiete |
| **WAULT** | Gewichtete Restlaufzeit der Mietverträge |
| **CapEx** | Capital Expenditure (Investitionen) |
